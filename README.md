# Five-Pillars
The Five Pillars: A simple, practical framework to check and support the foundations of your wellbeing.
This repository contains two theses and two checklists designed to help people understand and access the basics of mental and physical health.

Included Files
1. The Five Pillars Check - With Notes.pdf - Accompanies the framework thesis; a systems based check for foundational wellbeing.
2. The Five Pillars Check.pdf - Simple, emoji-friendly checklist for anyone to use.
3. The Five Pillars Core Framework.pdf - Detailed, formal explanation aimed at academic audiences.
4. The Five Pillars Practical Guide.pdf - Plain language version with examples and simplified explanations.

Purpose
The Five Pillars framework evaluates five essential ares: Food, Sleep, Connection, Expression, and Meaning. It is a tool for understanding fundamental needs, not a diagnostic or treatment guide.

How to Use
Read the appropriate thesis for context.
Use the corresponding checklist to quickly see which pillars may need attention.
Reflect, stabilize, and monitor your wellbeing over time.
